PaperSignals Parametrized

Building off of a simple box design interface by rahul at connectionlab, this tool allows you to input parameter dimensions and output a pdf. This is the first iteration which takes the microcontroller from the PaperSignals umbrella template, converts it to a svg instead of a pdf, and reshapes it based on the dimensions given by the user. This demonstrates that Papersignals templates can be Parametrized and therefore improved.
